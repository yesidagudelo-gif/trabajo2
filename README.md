# Trabajo Independiente 2: Historia de Steam con Float y Flexbox

## Descripción del Proyecto
Este proyecto es una práctica de diseño web donde construimos páginas sobre la historia de Steam utilizando dos técnicas de layout: Float (método tradicional) y Flexbox (método moderno). Cada página muestra la evolución de Steam desde su lanzamiento en 2003 hasta la actualidad, con contenido adaptado a diferentes estructuras de layout.

## 📌 ¿De qué trata este proyecto?

El objetivo es comparar:
1. **El método tradicional (Float):** Layout con floats, clears y cálculos manuales de espacios.
2. **El método moderno (Flexbox):** Layout automático con flexbox para distribución natural de elementos.

Cada folder representa una variante visual:
- **Folder 1:** Tema azul, layout de dos columnas (imagen/texto a la izquierda, texto largo a la derecha).
- **Folder 2:** Tema verde, layout con sección larga, tres cajas y una sección inferior.
- **Folder 3:** Tema rojo, layout de tres columnas con video integrado.

## 💡 Lo que nos enseña este ejercicio

Al comparar ambas técnicas:
- **Flexibilidad en alturas:** Float requiere forzar alturas iguales; Flexbox las ajusta automáticamente.
- **Espacios y centrado:** Float necesita cálculos precisos; Flexbox distribuye espacios naturalmente.
- **Mantenimiento:** Flexbox es más intuitivo y menos propenso a errores.
